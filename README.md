# swfbinarytag

Find binary tag by id from swf file. It's same as ```$ swfextract -b $id -o $output```.

## USAGE

```
$ swfbinarytag -input $file -id $id -output $output
```

## INSTALL

```
$ go get github.com/soh335/swfbinarytag/cmd/swfbinarytag
```
